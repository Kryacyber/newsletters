Cybersecurity Weekly: AI-Driven Insights on Emerging Threats and Major Disruptions

21 September, 2025

This week’s edition highlights critical disruptions in global infrastructure, emerging zero-click vulnerabilities, and sophisticated nation-state cyber campaigns. From airport check-in failures to a critical flaw in OpenAI’s Deep Research agent, attackers are leveraging new techniques to exploit sensitive data. Meanwhile, cloud security breaches, including exposed SonicWall backups and fake GitHub repositories distributing malware, underscore the persistent risks in enterprise environments. Nation-state actors continue to target cryptocurrency and retail sectors with malware-laced lures, while Microsoft and Fortra have patched severe vulnerabilities in identity and file transfer systems. Stay vigilant—these threats demand immediate attention.

General Security News

  • 2025-09-20 Cyberattack Disrupts Check-In Systems at Major European Airports

    A cyberattack forced major European airports to switch to manual check-in and boarding, disrupting operations for thousands of travelers.

  • 2025-09-19 Summer 2025 SOC 1 Report Available for 183 AWS Services

    AWS has released its Summer 2025 SOC 1 report, covering 183 services and demonstrating compliance with security controls over the past year.

  • 2025-09-19 AWS Organizations Now Supports Full IAM Language for SCPs

    AWS Organizations now allows full IAM policy language for Service Control Policies (SCPs), enabling more granular access controls.

  • 2025-09-19 U.K. Arrests Two Teen Scattered Spider Hackers Linked to TfL Cyber Attack

    Law enforcement arrested two teens from the Scattered Spider hacking group, suspected of participating in a 2024 cyberattack on Transport for London.

Data Breaches

  • 2025-09-20 ShadowLeak Zero-Click Flaw Leaks Gmail Data via OpenAI ChatGPT

    A zero-click vulnerability in OpenAI’s Deep Research agent allowed attackers to leak Gmail inbox data without user interaction.

  • 2025-09-18 SonicWall Urges Password Resets After Cloud Backup Breach

    SonicWall detected unauthorized access to firewall backup files in the cloud, affecting less than 5% of customers.

Malware

  • 2025-09-21 DPRK Hackers Use ClickFix Lures to Deliver BeaverTail Malware

    North Korean hackers targeted cryptocurrency and retail organizations with ClickFix-style lures to deliver BeaverTail malware.

  • 2025-09-20 LastPass Warns of Fake GitHub Repositories Distributing Infostealer Malware

    Fake GitHub repositories targeting macOS users distributed Atomic infostealer malware, posing as legitimate tools.

Vulnerabilities

  • 2025-09-22 Fortra Patches Critical GoAnywhere MFT Vulnerability

    A critical deserialization flaw in Fortra’s GoAnywhere MFT allowed command injection, posing a severe security risk.

  • 2025-09-22 Microsoft Patches Critical Entra ID Flaw Enabling Global Admin Impersonation

    A critical token validation failure in Microsoft Entra ID could have allowed attackers to impersonate Global Administrators.

Policy & Legal

No updates this week.

Key Takeaways

This week’s threats underscore the evolving tactics of cybercriminals and nation-state actors. Zero-click exploits and malware-laced lures highlight the need for proactive security measures. Organizations must prioritize patching critical vulnerabilities, monitoring cloud backups, and educating users about phishing risks. As infrastructure disruptions continue, resilience planning remains essential to mitigate future impacts.
