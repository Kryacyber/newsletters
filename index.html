Weekly Cybersecurity Digest: Critical Disruptions and Emerging Threats

This week’s top stories highlight critical disruptions in aviation cybersecurity, cloud security advancements, and emerging threats from state-sponsored actors. From zero-click vulnerabilities in AI tools to a critical flaw in Microsoft’s identity service, the week underscores the evolving risks in digital infrastructure and the need for proactive defense strategies.

General Security News

  • Cyberattack Disrupts Check-In Systems at Major European Airports 2025-09-20

    A cyberattack disrupted airport electronic systems, forcing European airports to switch to manual check-in and boarding processes.

    Collins Aerospace systems were targeted in a cyberattack, affecting passenger check-in, boarding pass printing, and luggage handling.

Policy & Legal

  • AWS Organizations Now Supports Full IAM Language for SCPs 2025-09-19

Data Breaches

    A zero-click vulnerability in OpenAI’s ChatGPT Deep Research agent could leak Gmail inbox data without user interaction.

Malware

    Fake GitHub repositories are distributing Atomic infostealer malware to macOS users, posing as legitimate tools.

Vulnerabilities

  • Microsoft Patches Critical Entra ID Flaw Enabling Global Admin Impersonation 2025-09-20

    Microsoft patched a critical flaw in Entra ID (Azure Active Directory) that could allow attackers to impersonate Global Administrators.

Conclusion

This week’s cybersecurity landscape reveals critical vulnerabilities in aviation systems, cloud services, and AI tools, along with coordinated attacks from state-sponsored actors. Organizations must prioritize proactive security measures, including patch management, threat intelligence, and employee awareness training to mitigate these evolving risks.
